Goedel.Utilities Namespace

The Goedel.Utilities namespace contains various utilities classes. These include
  • Methods of converting binary data to/from character strings using standard encodings.
  • Testing if a character is of a particular type.
  • Convenience routines to test various types of assertion and throw an exception using an exception factory method such as the ones created by Exceptional.
