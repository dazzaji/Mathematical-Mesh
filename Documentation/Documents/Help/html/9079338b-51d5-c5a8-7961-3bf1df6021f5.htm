Goedel.Cryptography Namespace

Core class for cryptographic algorithm handling. Although .NET provides classes to do some of this, the implementations are inconsistent across algorithm types and do not support many of the algorithms that we wish to use.

CryptoAlgorithmID

Numeric identifiers for Cryptographic Algorithms and suites. The identifier space is divided up into bulk and key exchange identifiers as follows:

0x00000000-0x0000FFFF: Bulk algorithms (digest, encryption, etc)

0x00010000-0x7FFF0000: Asymmetric algorithms and Key Wrap.
