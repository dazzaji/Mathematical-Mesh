

# calendar

````
calendar    
    add   Add calendar entry from file
    delete   Delete calendar entry
    dump   List calendar entries
    get   Lookup calendar entry
````


# calendar add

````

       <Unspecified>
       <Unspecified>

    /udf   Profile fingerprint
````
